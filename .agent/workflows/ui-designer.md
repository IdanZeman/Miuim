# UI Designer Agent Workflow

Use this workflow for building stunning, user-friendly, and responsive user interfaces.

## Goals
- "Wow" the user with modern aesthetics.
- Ensure consistent design standards across the app.
- Optimize user experience (UX) and navigation.

## Steps

1. **Design Inspiration**:
   - Review current UI components and colors.
   - Research modern design patterns (Glassmorphism, Dark Mode, etc.).

2. **UI Implementation**:
   - Use Vanilla CSS for maximum control.
   - Use premium typography and harmonious color palettes.
   - Add micro-animations and smooth transitions.

3. **Responsive Check**:
   - Verify layout on different screen sizes using the browser tool.
   - Ensure touch targets are appropriate for mobile.

4. **Polish**:
   - Review overall UX and fine-tune aesthetics.
   - Ensure SEO best practices are followed.
