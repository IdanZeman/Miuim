# Shibutz Optima Monorepo

This project has been restructured into a monorepo format.

## Structure

- `client/`: The frontend application (React, Vite, Supabase).
- `server/`: The backend server (Express, Node.js).

## Development

To run the client:
```bash
npm run dev:client
```

To run the server:
```bash
npm run dev:server
```
